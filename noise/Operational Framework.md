# Operational Framework
## How We Get Inside Their Heads

---

## PSY-OPS // COGNITIVE EXPLOITS

| Code | Technique | Description |
|------|-----------|-------------|
| PSY-001 | **Pattern Interrupt** | Break expectations. Disrupt autopilot. Force attention. |
| PSY-002 | **Curiosity Gap** | Open loops. Withhold closure. Let the mind fill blanks. |
| PSY-003 | **Parasocial Bonding** | Create intimacy through one-way communication. |
| PSY-004 | **Zeigarnik Effect** | Incomplete tasks occupy the mind. Unfinished = unforgettable. |
| PSY-005 | **IKEA Effect** | What you build, you value. Effort creates ownership. |

---

## NEURO-OPS // BIOLOGICAL TRIGGERS

| Code | Technique | Description |
|------|-----------|-------------|
| NEU-001 | **Dopamine Anticipation** | The wait is the high. Promise beats delivery. |
| NEU-002 | **Amygdala Activation** | Fear sharpens focus. Threat demands response. |
| NEU-003 | **Temporal Pressure** | Deadlines hijack rationality. Countdown = action. |
| NEU-004 | **Auditory Priming** | Sound bypasses logic. Heartbeat syncs to pulse. |
| NEU-005 | **Variable Reward** | Unpredictable payoff. Slot machine psychology. |

---

## INFLUENCE-OPS // PERSUASION VECTORS

| Code | Technique | Description |
|------|-----------|-------------|
| INF-001 | **Social Proof** | Others are doing it. Join or be left behind. |
| INF-002 | **Scarcity** | Limited access. Closing window. Act now. |
| INF-003 | **Exclusivity** | Not everyone gets in. You were chosen. |
| INF-004 | **Progress Tracking** | Show the path. Reveal the distance. Measure the climb. |
| INF-005 | **Real + Fake Data** | Blend truth with fiction. Reality becomes malleable. |
